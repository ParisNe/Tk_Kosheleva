## Краткое описание

WPF-приложение для решения:

- **Линейного уравнения:** `a·x + b = 0`
- **Квадратного уравнения:** `a·x² + b·x + c = 0`

Особенности:
- Выбор типа уравнения (RadioButton)
- Коэффициенты `a`, `b`, `c` (для квадратного `a` может быть 0)
- Валидация ввода (только числа, целые и дробные)
- Обработка особых случаев:
  - Бесконечно много решений
  - Нет решений
  - Комплексные корни (выводятся в формате `x = real + imag i`)
- Автоматизированные тесты (MSTest)
